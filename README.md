# Minimal Core
The base Minimal compiler on which the Minimal language extensions are added.

## Project Structure
- **cmd:** The entrypoints of the applications such as the compiler itself or the LSP
- **config:** TOML-files to change settings
- **docs:** The documentation of the compiler itself

- **built-in:** Some basic layers used for the Minimal language such as a lexer, parser...
